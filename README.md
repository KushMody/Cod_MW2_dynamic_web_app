# Call of Duty: MWII - Dynamic Web Experience

A high-fidelity, visually immersive landing page replica for **Call of Duty: Modern Warfare II**. This project demonstrates modern frontend techniques to create a cinematic user experience, featuring video backgrounds, complex scroll-based animations, and a premium "gamified" aesthetic.
